ПЗ №1

Рівень 1, завдання 1 — створити програму, яка виводить на екран числа від 1 до 10.

Рівень 2, завдання 1 — написати програму, яка знаходить середнє значення з трьох чисел, введених користувачем.

ПЗ №2

Рівень 1, завдання 5 — відображення зображення: створити HTML-файл з елементом <script>, використати fetch для завантаження JSON-файлу з URL зображення, створити елемент <img> з URL зображення та додати елемент <img> на веб-сторінку.

Рівень 2, завдання 5 — перевірка JSON: написати програму, яка перевіряє, чи даний текст є дійсним JSON. Використати вбудовані функції мови програмування або бібліотеки для перевірки JSON та вивести результат перевірки: дійсний JSON або недійсний JSON.

ЛБ №1

Рівень 1, завдання 5 — написати сайт курсу валют, у якого буде можливість додавати нові курси співвідношення валюти до гривні. Для кожної валюти має бути два значення: курс купівлі та курс продажу. У результаті потрібно отримати таблицю з валютами, яка має 3 колонки: назву валюти, курс купівлі та курс продажу.

Рівень 2, завдання 5 — додати можливість збереження результатів поточного дня в окремій вкладці.

ЛБ №2

Рівень 1, завдання 5 — створити простий каталог курсів на Node.js з використанням Express. Реалізувати frontend на React для відображення курсів.

Рівень 2, завдання 5 — додати можливість запису на курс. Реалізувати систему відгуків та рейтингу курсів.
